## 🚀 English Introduction

### The Future of Research is Here. And It Never Sleeps.

**AutoResearchCodebaseWithHarness** is not just another ML framework. It's a **fundamental shift** in how research gets done.

Imagine this: You describe a hypothesis. Go to sleep. Wake up to **50+ experiments completed**, each one carefully evaluated, the best kept, the rest discarded. All documented. All reproducible. All in Git history.

This is **autonomous research at scale**—powered by Rust's performance, guided by AI's intelligence, and constrained by resource limits that prevent catastrophic failures.

### What Makes It Revolutionary

**🔬 Full-Stack Autonomy**
- Not just "run this script 100 times"
- **Understands your codebase** through AST parsing and dependency graphs
- **Generates informed hypotheses** based on code patterns and failure analysis  
- **Designs experiments** with proper isolation and resource constraints
- **Makes decisions** to keep or discard based on quantified metrics

**⚡ Rust Performance + Safety**
- Zero-cost abstractions mean your experiments run at maximum speed
- Memory safety guarantees prevent entire classes of bugs
- Fearless concurrency for parallel experiment execution
- Type system catches errors at compile time, not 3 hours into training

**🎯 Harness Architecture**
- **Resource Management**: Automatic VRAM/CPU/memory monitoring
- **Time Budgets**: Fixed 5-minute experiments (or configurable)
- **Isolation**: Experiments run in separate branches with automatic cleanup
- **Crash Recovery**: OOM? Timeout? Logged and moved on

**🧠 Codebase Intelligence**
- Parse and understand code structure
- Track dependencies between modules
- Predict blast radius of changes
- Quantify complexity (`simplicity_score`)

**🤖 Multi-Model Collaboration**
- **Claude Code**: Fast execution, tool use, codebase navigation
- **GPT-5.4** (via Codex): Deep reasoning, literature review, architectural critique
- Each model does what it's best at

**📊 Simplicity-First Design**
- Complexity is technical debt. We quantify it.
- Equal performance? Simpler code wins automatically.
- 0.001 improvement + 50 LOC complexity? Rejected.
- Equal performance - 30% code? Major win. Keep it.

**ML Researchers** who want to explore hyperparameter spaces overnight

**Startups** who need to move fast and find optimal solutions  
**Anyone** who's tired of babysitting experiments

**That's 36× faster iteration.** That's the difference between a week and a month. Between catching up and leading the field.
