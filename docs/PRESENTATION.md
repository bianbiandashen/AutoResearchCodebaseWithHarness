## 🔧 如何修改

### 1. 修改内容

编辑 `generate-presentation.js`，找到对应的页面代码：

```javascript
// 例如：修改封面标题
slide1.addText("你的新标题", {
  x: 0.5, y: 1.8, w: 9, h: 0.8,
  fontSize: 40,
  bold: true,
  color: colors.white,
  align: "center",
  fontFace: "Microsoft YaHei"
});
```

### 2. 更换配色

修改 `colors` 对象：

```javascript
const colors = {
  primary: "1E2761",    // 改成你想要的颜色
  secondary: "CADCFC",
  // ...
};
```

**推荐配色**（参考 pptx-cn/color-schemes.js）：
- 商务深蓝：`businessBlue`
- 健康青色：`healthTeal`
- 科技紫：`techPurple`（当前）
- 活力橙红：`coralEnergy`

### 3. 重新生成

```bash
cd docs
node generate-presentation.js
```

新的 PPT 会覆盖旧文件。

## 🖼️ 导出图片

如果需要将 PPT 转换为图片（用于 README 或社交媒体）：

### macOS/Linux

```bash
# 安装 LibreOffice（如果未安装）
brew install libreoffice  # macOS
# 或 apt install libreoffice  # Linux

# 转换为 PDF
soffice --headless --convert-to pdf "AutoResearchCodebaseWithHarness-爆款介绍.pptx"

# PDF 转图片（需要 poppler）
brew install poppler  # macOS
pdftoppm -jpeg -r 150 AutoResearchCodebaseWithHarness-爆款介绍.pdf slide

# 结果：slide-1.jpg, slide-2.jpg, ...
```

### Windows

```powershell
# 使用 PowerPoint 导出
# 打开 PPT → 文件 → 导出 → 更改文件类型 → PNG/JPEG
```
